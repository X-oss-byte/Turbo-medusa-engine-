# Features and Guides

Medusa is an ecommerce engine that offers ecommerce features essential to all types of businesses. Starting from the basic products management and order management, to the more advanced automated return and exchange flows.

This part of the documentation covers Medusaâ€™s ecommerce features and how they can be used. It also includes guides that help developers utilize these features to create unique digital experiences.
